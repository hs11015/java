메소드는 클래스 안의 함수를 부르는 말.

자바는 클래스의 기반. 그래서 자바에서는 '함수'가 아닌 메소드로 자주 부름

​

System은 자바에 내장되어 있는 표준 입출력클래스.

System이 클래스이기 떄문에 클래스 안에 저장되어 있는 메소드(함수)와 변수들이 나열됨.

오늘 사용한 메소드는 "System.out"이고 출력을 의미함.

​

System.out(출력) 종류

System.out.print : 문장을 출력

System.out.println : 문장을 출력한 후 마지막에 줄바꿈이 자동으로 되는 메소드

System.out.printf : 문장 안에 지시자를 사용할 수 있고, 쉼표로 구분함.

​

지시자

%d : (decimal) 정수 형식으로 출력

%f : (floating-point) 실수 형식으로 출력

%c : (character) 문자 형식으로 출력

%s : (string) 문자열 형식으로 출력

%b : (boolean) 논리 형식으로 출력
